# Cloud Task Manager

A cloud-native task management API built to demonstrate DevOps practices.

## Tech Stack

- Python Flask
- Git
- Docker
- GitHub Actions
- AWS
- Kubernetes
- Terraform

## Current Features

- Create tasks
- View tasks
- Delete tasks

## Future Improvements

- Docker containerization
- CI/CD pipeline
- Cloud deployment
- Kubernetes orchestration
- Monitoring

## CI/CD Pipeline

Implemented GitHub Actions workflow that:

- Automatically triggers on code push
- Installs dependencies
- Builds Docker image
- Validates application packaging
